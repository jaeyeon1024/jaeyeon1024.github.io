LP — 선형 계획법

🥢 점심 메뉴 고르기 LP

학교 앞 분식집에서 떡볶이와 김밥을 몇 인분씩 먹을지 결정. 포만감을 최대화하되 예산과 최대 섭취량 제한을 지켜야 한다.

떡볶이김밥
가격4,000원2,000원
포만감4점3점
최소1인분1줄
최대2인분3줄
결정변수
x₁ = 떡볶이 인분 수
x₂ = 김밥 줄 수

목적함수
max Z = 4x₁ + 3x₂

제약조건
4000x₁ + 2000x₂ ≤ 14,000
x₁ ≤ 2, x₂ ≤ 3
x₁ ≥ 1, x₂ ≥ 1
x₁, x₂ ≥ 0
✓ 최적해: 떡볶이 2인분, 김밥 3줄 → 포만감 17점

NLP — 비선형 계획법

🚴 사이클 페달링 NLP

실내 자전거 운동 시 페달 속도(x₁)와 저항 강도(x₂)를 설정. 인체 근육은 특정 구간에서 리듬이 맞을 때 효율이 최대가 되는 sin 곡선 특성을 가진다.

페달 속도저항 강도
칼로리 소모10·sin(πx₁/6)8·sin(πx₂/6)
심박수 부하2x₁ bpm3x₂ bpm
범위1 ~ 5단계1 ~ 5단계
결정변수
x₁ = 페달 속도 단계
x₂ = 저항 강도 단계

목적함수 (비선형)
max Z = 10·sin(πx₁/6) + 8·sin(πx₂/6)

제약조건
2x₁ + 3x₂ ≤ 20
1 ≤ x₁ ≤ 5, 1 ≤ x₂ ≤ 5
x₁, x₂ ≥ 0
✓ 최적해: 속도 3단계, 강도 3단계 → 칼로리 소모 18 kcal

SO — 확률적 최적화

🎫 공연 티켓 준비 SO

공연장 운영자가 일반석(q₁)과 VIP석(q₂) 티켓을 몇 장씩 준비할지 결정. 실제 관객 수요 D는 공연 당일까지 알 수 없다. 수요를 확률 분포로 모델링해 기댓값을 최적화한다.

시나리오수요확률
수요 적음50명0.3
수요 보통80명0.5
수요 많음110명0.2
결정변수
q₁ = 일반석 수량, q₂ = VIP석 수량

목적함수
max E[수익]
= 0.3·수익(q,50)
+ 0.5·수익(q,80)
+ 0.2·수익(q,110)

제약조건
q₁ + q₂ ≤ 100
q₁ ≥ 40, q₂ ≥ 10
✓ 최적해: 일반석 60장, VIP석 40장 → 기대 수익 1,610,000원

MO — 다목적 최적화

💻 노트북 구매 MO

CPU 성능 등급(x₁)과 배터리 용량 등급(x₂)을 결정. 성능을 높이면 배터리가 줄고, 배터리를 높이면 가격이 오른다. 세 목표가 서로 충돌하므로 단일 최적해가 없다.

목적함수방향수식
가격최소화20x₁ + 10x₂
성능최대화10·x₁^1.2 + 3x₂
배터리최대화−1.5x₁ + 9x₂
목적함수 (3개)
min f₁ = 20x₁ + 10x₂
max f₂ = 10·x₁^1.2 + 3x₂
max f₃ = −1.5x₁ + 9x₂

제약조건
20x₁ + 10x₂ ≤ 150
1 ≤ x₁ ≤ 5, 1 ≤ x₂ ≤ 5
✓ 단일 최적해 없음 — Pareto front에서 균형점 선택
